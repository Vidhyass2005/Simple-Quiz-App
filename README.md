## ğŸ® **How to Play**

### **Step 1: Setup**
1. Select a **category** (Geography, Science, History, Entertainment, Sports, or Mixed)
2. Choose **number of questions** (3, 5, or 10)
3. Click **"Start Quiz"**

### **Step 2: Play**
1. Read the question carefully
2. Select an answer by:
   - Clicking the option button, OR
   - Pressing **1, 2, 3, or 4** on your keyboard
3. Watch the **30-second timer** - answer before time runs out!
4. Get instant visual feedback
5. Click **"Next Question"** to continue

### **Step 3: Results**
1. View your **score percentage**
2. See detailed **statistics** (correct/wrong/skipped)
3. Read **personalized feedback** message
4. Click **"Try Again"** for another round
